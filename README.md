# 💰 FinanceAI - AI Driven Personal Finance Tracker

A full-stack personal finance management web application with AI-powered insights, expense tracking, budget management, and calendar view.

---

## ✨ Features

- JWT Authentication — Secure login and signup
- Dashboard — Spending overview with charts
- Expense Tracking — Add, edit, delete expenses
- Budget Management — Set spending limits per category
- Calendar View — View expenses by date
- AI Insights — Spending analysis and recommendations
- Responsive Design — Works on desktop and mobile

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, shadcn/ui, Recharts, React Router, Axios

**Backend:** Python, FastAPI, MySQL, JWT Authentication

---

## 🚀 How to Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm start
```

Open http://localhost:3000
